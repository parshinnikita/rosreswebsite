# Сайт компании ООО «Техресурсы»

Редизайн сайта компании ООО «Техресурсы» (rosres.ru) с использованием дизайна, вдохновленного сайтом eurochem.ru.

## Структура проекта

```
rosreswebsite/
├── css/
│   ├── normalize.css
│   └── styles.css
├── js/
│   └── main.js
├── images/
│   ├── logo.png
│   ├── logo-white.png
│   ├── slider1.jpg
│   ├── news1.jpg
│   └── partners/
│       ├── partner1.png
│       ├── partner2.png
│       └── ...
├── index.html
└── README.md
```

## Требуемые изображения

Для полноценной работы сайта необходимо добавить следующие изображения:

1. `images/logo.png` - логотип компании для светлого фона
2. `images/logo-white.png` - логотип компании для темного фона (футер)
3. `images/slider1.jpg` - изображение для главного слайдера
4. `images/news1.jpg` - изображение для новости
5. `images/partners/partner1.png` - `images/partners/partner5.png` - логотипы партнеров

## Установка и запуск

1. Скачайте или клонируйте репозиторий
2. Добавьте необходимые изображения
3. Откройте файл `index.html` в браузере для просмотра

## Особенности дизайна

- Современный, чистый дизайн в стиле eurochem.ru
- Адаптивная верстка для всех устройств
- Основные цвета: синий (#005baa) и зеленый (#00a650)
- Шрифт: Roboto (Google Fonts)
